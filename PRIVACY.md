# Privacy Policy — Eclyzer Hub Browser Extension

**Last updated:** June 1, 2026

---

## Overview

Eclyzer Hub is a browser extension that provides quick-access shortcuts to Eclyzer services. It functions solely as a launcher — opening links in new browser tabs when clicked by the user.

---

## Data Collection

**Eclyzer Hub does not collect any data.**

Specifically, this extension:

- Does **not** collect personal information
- Does **not** collect browsing history
- Does **not** read or modify web page content
- Does **not** access browser tabs, bookmarks, or cookies
- Does **not** store any data locally or remotely
- Does **not** use cookies or local storage
- Does **not** transmit any data to external servers

---

## Permissions

Eclyzer Hub requests **zero browser permissions**. The extension operates with an empty permissions array (`"permissions": []`), which means it has no access to any browser APIs beyond the default popup functionality.

---

## Third-Party Services

Eclyzer Hub does **not** use any third-party services, including:

- No analytics (Google Analytics, Mixpanel, etc.)
- No advertising networks
- No crash reporting services
- No external CDNs or scripts
- No tracking pixels

---

## Network Requests

Eclyzer Hub makes **no network requests**. The extension runs entirely offline. When a user clicks a button, the browser opens the corresponding URL in a new tab using standard browser navigation — no data is sent or received by the extension itself.

---

## How It Works

1. The user clicks the Eclyzer Hub icon in the browser toolbar
2. A popup appears showing shortcut buttons to Eclyzer services
3. When the user clicks a button, a new tab opens with the selected URL
4. No data is collected, stored, or transmitted at any point

---

## External Links

The extension opens the following URLs when clicked by the user:

- `https://eclyzer.net` — Eclyzer main website
- `https://ai.eclyzer.net` — Eclyzer AI assistant
- `https://eclyzer.net/markdown` — Eclyzer Markdown Studio tool
- `https://github.com/hisyamnurx` — GitHub profile

These are official Eclyzer properties. The extension does not redirect to any other domains.

---

## Children's Privacy

Eclyzer Hub does not knowingly collect any information from children under the age of 13, as it does not collect any information from any user.
